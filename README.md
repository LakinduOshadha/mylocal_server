# mylocal_service
-- Backend service for mylocal ui --

How to run:

create .env file\
Add

    ENTS_BASE_URL =
    CENSUS_BASE_URL =

    # API settings
    API_HOST = '0.0.0.0'
    API_PORT = 4000

In terminal

    pip install -r requirements.txt
    python mylocal_service.py
